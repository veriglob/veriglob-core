# Governance

Veriglob is an open, community-driven project.

---

## Decision Making

- Protocol changes require discussion
- Breaking changes require consensus
- Security decisions prioritize safety over speed

---

## Maintainers

Maintainers are responsible for:

- Reviewing PRs
- Managing releases
- Handling security disclosures

---

## Community Values

- Privacy by default
- User sovereignty
- Interoperability
- Open standards
