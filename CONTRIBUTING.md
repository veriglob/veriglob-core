# Contributing to Veriglob

First of all, thank you for considering contributing to Veriglob ❤️  
This project exists to build open, decentralised, privacy-preserving identity infrastructure for everyone.

---

## Ways to Contribute

You can contribute by:

- Reporting bugs
- Proposing features or DID methods
- Improving documentation
- Writing tests
- Reviewing pull requests
- Building SDKs or integrations

---

## Development Setup

### Requirements

- Go (latest stable)
- Git
- Basic understanding of DID & cryptography

### Clone the repo

```bash
git clone https://github.com/veriglob/veriglob-core.git
cd veriglob-core
```
